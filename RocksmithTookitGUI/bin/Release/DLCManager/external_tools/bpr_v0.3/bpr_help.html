Introduction

BPR is a tool to facilitate quick updates on XML files included in RS packages.

With BPR you can only :

  • Unpack and repack a package (need a valid RSToolkit install)
  • Add or remove DD on a track (need a valid RSToolkit install with DDC)
  • Modify rhythm informations (measures, beats, time signature, bpm)
  • Modify sections and phrases
  • Adjust Score Attack levels

For all other kind of modifications you need to use EOF.

If the CDLC is totally out of sync (notes are not in sync with the soundtrack), don't waste your time with this tool ! You should recreate the CDLC from scratch ...

BPR known limitations for the beta release are :

  • Only for Windows 32 bit compatible systems
  • Works only with RS packages for PC or Mac
  • There is no undo and redo functionality

How to set BPR configuration ?

Before starting to use BPR, you have to set its properties.

Either you can update the "bpr_config.xml" file, either you can use the [Edit][Properties] menu.

How to unpack and repack RS CDLCs ?

To unpack a package, simply use [File][Unpack] menu and choose the "psarc" file.

Once you have unpacked several "psarc" files, you can change the current folder and the current track by using the two list items under the main menu.

To pack the current folder, use [File][Pack] and choose file name and location.

To delete the current folder (which contains the result files of an unpacked package) use [File][Delete Folder] menu.

How to navigate in the song ?

You can click on beats, phrases and sections in the "Sections & Phrases" area. The corresponding part of the tablature will be displayed.

When the tablature canvas is selected, you can use :

  • [Left] and [Right] keys to navigate.
  • [Ctrl+Left] and [Ctrl+Right] combos to navigate quicker.
  • [Home] key to go to the first measure.
  • [End] key to go to the last measure.

How to select a beat ?

To select a single beat or a single measure, just [Left-Click] on it. Contextual informations are displayed automatically.

To select a range of measures :

  • Select the first measure with [Left-Click].
  • Select the last measure with [Shift]+[Left-Click].
  • Select an action in the menu that you want to apply on this range.

You can also use the following shortcuts :

  • [Ctrl+A] to select all measures.
  • [Ctrl+R] to select the rest of measures from the current one to the end.
  • [Ctrl+N] to select none.

How to move measures and beats ?

To move a vertical bar, you have to use the [Ctrl] key in combination with the mouse action.

To move a vertical bar and translate all following beats during the move, you have to use the [Alt] key instead.

When you release a beat on a gem, the beat is automatically adjusted on the center of the gem and the vertical bar is colored in blue.

To set a new BPM :

  • Select the first measure for this new BPM.
  • Set the new value in the "BPM" field.
  • Use [Enter] to validate.

To change the number of beats in a range of measures :

  • Select the range of measures you want to update.
  • Set the new value in the "Time Signature" field.
  • Use [Enter] to validate.

You can also use the [Measure][Add], [Measure][Delete], [Beat][Add] and [Beat][Delete] in the menu.

How to modify sections and phrases ?

To add a new section :

  • Select the first beat of a measure.
  • Set the "Section" field with a valid name (intro, verse, chorus, bridge, interlude, outro, ...).
  • Use [Enter] to validate.

To delete an existing section :

  • Select the first measure of the section.
  • Use the [Section][Delete] menu.

You can also merge two following sections with the same name by using [Section][Merge] on the second occurrence.

Use [Section][Split] on an existing phrase to split the current section.

Use [Phrase][Add] or [Phrase][Delete] to add or delete a phrase (phrases are automatically numbered to avoid having to set their name).

Use [Phrase][Move Left] or [Phrase][Move Right] to move the selected phrase one measure left or one measure right.

How to resync an existing CDLC ?

The global process to resync a CDLC is :

  • Find a good tab to do this manual work.
  • Unpack the CDLC.
  • Synchronize the main measures. Secondary beats are automatically evenly spaced inside each measure.
  • When you did the job for all main measures, [Edit][Select All] and use the [Beat][Adjust] to center each secondary beat with its own overlaped gem.
  • Define the sections and phrases.
  • Use [File][Save] to save the XML file without DD or [File][Save and Generate DD] to save the XML file with DD inside.
  • Use [File][Transfer] to transfer the modifications (beats, sections and phrases) on another arrangement of the same CDLC.
  • Check and adjust the other arrangement tracks.
  • Pack the CDLC.
  • Try it under RS.

How to adjust Score Attack levels ?

The global process to ajust Score Attack levels is :

  • Open a track.
  • If the track hasn't DD inside, generate it with [File][Save and Generate DD].
  • Select the first beat of a phrase.
  • Update the fields "Score Attack Easy", "Score Attack Medium" and "Score Attack Hard".
  • Hit the [Enter] key to validate the modifications (all similar phrases are updated with the same levels).
  • Do this for all other phrases.
  • Save the track with [file][Save Score Attack Updates].
  • Pack the CDLC.
  • Try it under RS.

You can visualize similar phrases by using [Phrase][Show Similar Phrases]. All phrases with the same color have the same sequence of notes.

As the colors are chosen randomly, you can generate a new set of colors by using again the same action. Use [Phrase][Hide Similar Phrases] to hide the color map.
