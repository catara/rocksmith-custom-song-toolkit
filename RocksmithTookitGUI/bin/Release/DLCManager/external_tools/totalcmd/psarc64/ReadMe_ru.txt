psarc v1.3 - wcx plugin for Total Commander.

Архиваторный плагин, позволяющий работать с архивами формата PSARC
(просмотр/распаковка/создание), который используется во многих играх
от игровой консоли Sony PlayStation 3 (PS3).
Встроенные алгоритмы сжатия ZLIB и LZMA.
Поддерживает мультиядерность для ускорения работы.

Как инсталлировать плагин в Total Commander:
1. Дважды щёлкнуть мышкой по скачанному zip файлу.
2. Следовать инструкциям.

Что нового в версии v1.4:

+ Реализована обратная совместимость с Windows XP.

+ Устранен баг, приводящий к краху при попытке открыть PSARC с неизвестным
  методом сжатия, например из игры "PlayStation All-Stars Battle Royale".

-------------------------------------------------------------------------------
Что нового в версии v1.3:

+ В диалоговом окне настроек плагина для создания нового файла архива
  добавлена опция "Absolute Path Names". Она позволяет выбирать абсолютные
  (начинающиеся с символа /) или относительные пути для файлов, помещаемых
  в архив.
  Устанавливайте этот параметр таким же, каким он был в оригинальном архиве!
  Этот параметр для файла архива можно посмотреть в диалоговом окне
  "Свойства фала" Windows на закладке "PSARC" в строке "Path Names".

-------------------------------------------------------------------------------
Что нового в версии v1.2:

+ В дополнение к ZLIB добавлен алгоритм сжатия LZMA.
  При распаковке алгоритм сжатия определяется автоматически, при создании
  нового архива - выбирается в диалоговом окне настроек плагина.

+ Добавлена функция поиска и объединения одинаковых по содержимому файлов
  при создании архива. Таким образом в архив помещается только одна копия
  данных. При распаковке, естественно, все файлы будут на месте.
  Эта функция выбирается в диалоговом окне настроек плагина.

+ В стандартное диалоговое окно "Свойства фала" Windows добавлена закладка
  "PSARC", на которой отображается информация о версии архива, алгоритме
  сжатия, игнорировании регистра букв при подсчете контрольной суммы для
  имен файлов и количестве файлов в архиве.
  Эта закладка будет отображаться только для файлов psarc архивов.
  (Окно "Свойства файла" можно открыть через контекстное меню, щелкнув
   правой кнопкой мышки по файлу и выбрав в раскрывшемся меню пункт
   "Свойства" или "Properties".)

+ В диалоговом окне настроек плагина для создания нового файла архива
  добавлена опция "PSARC Version". Она позволяет выбрать версию psarc
  от 1.0 до 1.9, но фактически ни на что не влият. Эта информация
  записывается только лишь в заголовок вновь создаваемого архива.
  Рекомендуется устанавливать такой же номер версии, какой был
  в оригинальном файле архива.

+ Добавлена возможность работы в фоновом режиме.
