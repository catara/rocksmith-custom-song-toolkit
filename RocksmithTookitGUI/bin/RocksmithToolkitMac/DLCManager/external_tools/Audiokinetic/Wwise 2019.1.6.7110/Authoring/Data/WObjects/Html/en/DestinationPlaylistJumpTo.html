Jump To

Determines the entry point of a Music Playlist if the destination object of a Music Transition is a Music Playlist Container. You can choose from the following options:

  • Start of Playlist: The Music Playlist will start at the beginning.
  • Specific Playlist Item: The Music Playlist will start at the indicated Playlist Item.
  • Last Played Segment: The Music Playlist will start with the last played Music Segment (see Note).
  • Next Segment: The Music Playlist will start with the next Segment following the last played Music Segment (see Note).

Note: The options Last Played Segment and Next Segment only apply for transitions of the parent Music Switch Container during the same playback instance.

Default value: Start of Playlist
